# Handloom Artisan Platform

A PHP & MySQL based web application connecting artisans and customers.

## Features

- Customer Registration/Login
- Artisan Registration/Login
- Product Management
- Shopping Cart
- Checkout & Payment
- Order Tracking
- Admin Dashboard
- Stock Management
- Product Reviews
- Multiple Product Images
- Order Status Updates

## Technologies

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP
